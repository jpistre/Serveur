[description]: # (Différentes options pour le framework web en fonction des besoins)

Si l'objectif du site est d'être mis en oeuvre rapidement,simple et avec peu de complexité métier alors les options suivantes s'offrent à vous:

-Le framework Flask qui utilise le langage Python et qui est léger,minimaliste et très flexible.

-Le framework Express.js qui utilise le langage JavaScript(Node.js) et qui est simple,performant avec un large écosystème NPM et est à la base de la majorité des apps Node.

-Le framework Slim qui utilise le langage PHP et qui est minimaliste et efficaces pour de petites API REST.

-Les frameworks Gin ou Fiber qui utilisent le langage Go et qui sont très rapides et ont d'excellentes performances.


Ces différents frameworks sont sont parfaits pour des prototypes et des API simples.

Si l'objectif est d'avoir un site productif avec une structure sans lourdeur alors voici les frameworks recommandés:

-FastAPI qui utilise le langage Python et qui est rapide,moderne et avec un typage fort,parfait pour les API REST ou backends modernes

-NestJS qui utilise le langage JavaScript/TypeScript
