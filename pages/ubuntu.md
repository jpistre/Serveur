[description]: # (Installation et configuration de Ubuntu Server)

Télécharger l’image ISO Ubuntu Server 24.04 LTS en se rendant sur le site officiel:https://ubuntu.com/download/server

Mettre l'image ISO sur une clé USB et insérer la clé USB dans la machine qui deviendra le serveur.

Paramètrer la machine pour qu'elle boot sur la clé USB et redémarrer la machine. Au bout de quelques instants des messages tels que cela devraient apparaître:

![boot](https://ubuntucommunity.s3.us-east-2.amazonaws.com/original/2X/1/17ee449b2bd7c530d2f996215407fca5b722dcb2.png)

Après cela un menu de sélection de langage apparaîtra:

![langues](https://ubuntucommunity.s3.us-east-2.amazonaws.com/original/2X/e/e1d75e3584b6a3c23da39263fbf2f9ba6411de9a.png)

Une fois la langue choisie il vous sera demandé de choisir votre configuration de clavier:

![config clavier](https://ubuntucommunity.s3.us-east-2.amazonaws.com/original/2X/5/5c918dc341d92f647d6f1665ed2714922d5e688c.png)

Après cela il vous sera demandé de choisir le type d'installation que vous souhaitez, choisissez Ubuntu:

![install](https://ubuntucommunity.s3.us-east-2.amazonaws.com/original/2X/7/7e92f38c56ca7b2fbb8323d51d6ee35457fe4fce.png)

L'installateur devrait ensuite automatiquement détecter et essayer de configurer les connexions réseaux via DHCP.

Il vous faudra ensuite décider de la répartition des données et de choisir le disque sur lequel elle se fera.

Une fois ce choix fait un récapitulatif apparaîtra et une fois que vous aurez décidé de passer à la suite un message de confirmation apparaîtra à l'écran:

![confirmation](https://ubuntucommunity.s3.us-east-2.amazonaws.com/original/2X/c/c33fb0d6b4b9bfcb982eda42b21ced5187c38e7b.png)

Prenez bien le temps de vous assurer que tout est en ordre car une fois que vous aurez confirmé il n'y aura plus de retour arrière possible.

Une fois que vous aurez confirmé il vous sera demandé de renseigner quelques informations sur le serveur avec,notamment,son nom,un login,et un mot de passe:

![infos](https://ubuntucommunity.s3.us-east-2.amazonaws.com/original/2X/2/22deb8bda1b7df8dc94a19f6a23bb07c5b7664b0.png)

Après cela l'installation va commencer et une fois qu'elle sera terminée le message suivant apparîtra:

![install over](https://ubuntucommunity.s3.us-east-2.amazonaws.com/original/2X/c/c36b4880b5e4291ca57e6a51b527c05449fec48e.png)
